# 🌸 BookBloom
BookBloom is a Personalized-Manually Operated Digital Library built using HTML, CSS, and JavaScript.  
It is your own cozy Digital Bookshelf to collect books, cherish stories, & track your reading journey. It allows users to organize, review, and manage their books in an elegant and interactive interface.  

## Version: 1.0
## About  
BookBloom is a simple, front-end book tracker that lets you log the books you're reading, have read, or want to read — complete with cover images, ratings, reviews, and favorites — all stored right in your browser.  
## ✨ Features

- 📚 Add, edit, and delete books with title, author, genre, cover image, status, rating, and review
- ✏️ Edit book details
- 🗑️ Delete books
- 🗂️ Category filter by genre
- ❤️ Favorites — mark books as favorites, highlighted with a gold border
- ⭐ Star ratings from 1–5
- 📝 Add personal reviews
- 📖 Track reading status (To Read, Reading, Completed)
- ↕️ Sort by title (A–Z), author (A–Z), or highest rating
- 🔍 Search across title, author, genre, status, and review text
- 📊 Dashboard showing total books, favorites, average rating, and books completed
- 🎯 Reading goal progress bar tracking completed books against a 20-book goal
- 🌙 Dark mode toggle with persistent theme (saved in local storage)
- 💾 Local storage persistence — your library is saved in the browser, no backend required

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Local Storage API

## Getting Started
 
1. Download or clone the project files.
2. Make sure the `Images/` folder (with `MAIN-PAGE.jpeg`, `HERO.jpeg`, `FORM.jpeg`, `LIBRARY.jpeg`, `DASHBOARD.jpeg`) sits alongside `index.html`.
3. Open `index.html` in any modern browser — no build step or server needed.
## Usage
 
- Fill out the form and click **Add to Library** to save a book.
- Click **Edit** on any book card to load it back into the form for updating.
- Click **Delete** to remove a book (with a confirmation prompt).
- Click the heart icon to toggle a book as a favorite.
- Use the search bar, category dropdown, or sort dropdown to organize your library.
- Toggle **🌙 Dark Mode** in the top-right corner anytime.
