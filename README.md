# 🌸 BookBloom

BookBloom is a personal digital library built using HTML, CSS, and JavaScript. It allows users to organize, review, and manage their books in an elegant and interactive interface.

## ✨ Features

- 📚 Add new books with cover images
- ✏️ Edit book details
- 🗑️ Delete books
- ❤️ Mark books as favourites
- ⭐ Rate books
- 📝 Add personal reviews
- 📖 Track reading status (To Read, Reading, Completed)
- 🔍 Search books instantly
- 📊 Dashboard with library statistics
- 💾 Local Storage support

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Local Storage API
